DOOM CLOCK Screensaver
=====================

Installation:
1. Run install.bat as Administrator (Right-click -> Run as Administrator)
2. Screensaver settings will open automatically
3. Select "Doom Clock" from the list
4. Click "Apply"

To test: Double-click "Doom Clock.scr"

Controls:
- Move mouse or press any key to exit

Uninstall:
- Delete "Doom Clock.scr" from C:\Windows\System32\
- Delete "doom2016left.ttf" from C:\Windows\Fonts\
